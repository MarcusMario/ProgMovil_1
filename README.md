# Android Cero - Primer App

Aplicación móvil básica desarrollada en Android Studio como introducción al desarrollo de aplicaciones Android utilizando Kotlin.

---

## Descripción

Este proyecto consiste en la creación de una aplicación Android desde cero sin actividad inicial, configurando manualmente los componentes principales del proyecto. Se implementa una actividad (`MainActivity`) que carga una interfaz gráfica definida en XML, mostrando un mensaje centrado en pantalla.

---

## Características

- Creación de proyecto sin Activity inicial  
- Configuración de Gradle con Kotlin DSL  
- Compatibilidad con Java 11  
- Implementación de `MainActivity`  
- Uso de `ConstraintLayout`  
- Uso de recursos con `strings.xml`  
- Activación de ViewBinding  

---

## Tecnologías

- Kotlin  
- Android Studio  
- Gradle (Kotlin DSL)  
- XML  

---

## Objetivo

Comprender la estructura base de una aplicación Android, incluyendo la relación entre lógica (Kotlin), interfaz (XML) y configuración del proyecto.

---

## Resultado

La aplicación muestra en pantalla el texto:

"Hola Android"

centrado en la interfaz.
